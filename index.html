HTML

  • HTML defines the structure of a website
  • HTML information in the 'head' tag contains the metadata, which for the most part is not seen by users
  • Visible HTML elements need opening and closing tags
  • Levels of headings should decrease as you go down the page
  • Alt text must be provided to support assistive technology

Git

  • GitHub stores code, but also allows you to collaborate with other developers
  • Default branches in GitHub are called 'main'
  • Feature branches are used to track changes made to the main branch
  • Commits describe the changes made to feature branches
  • Changes to code made locally are pushed up to feature branches on GitHub, before being merged through pull requests
